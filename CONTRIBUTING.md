# Contributing a poem

Everyone is welcome here.

## How to submit

1. Fork github.com/IQTIA/queerdo-poetry-repo
2. Create a file in poems/ folder
3. Open a pull request

Not comfortable with Git? Email production@queerdos.eu

## Rules

Your words. Any language. Any length.
Content notes encouraged for heavy material.
No hate speech. No harassment.
